# Team Agency Portfolio

## Project Overview

**Team Agency Portfolio** is a professional, responsive, and data-driven portfolio website developed as part of the **Track A: Dev Agency Distributed Workflow** project.

The website acts as a central hub for the development team, introducing team members, showcasing skills, and providing access to all team projects hosted in separate repositories.

The application demonstrates collaboration using GitHub feature branches, pull requests, code reviews, and component ownership similar to a real-world software development agency.

## Git Workflow

### Main Branch

```bash
main
```

### Feature Branches

```bash
feature/header-hero

feature/team-skills

feature/projects

feature/contact-footer
```

### Development Process

1. Create feature branch
2. Develop assigned module
3. Commit changes
4. Push branch to GitHub
5. Create Pull Request
6. Peer Review
7. Merge into Main Branch
